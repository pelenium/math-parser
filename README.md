# math parser

## What can it parse?

### Basic math operations

```lisp
(+ 10 (/ 5 2)) ;; result - 12
```

```lisp
(/ 138 (+ 5 (* 3 6))) ;; result - 6
```

### Trigonometric functions and working with roots

```lisp
(sin (+ 3 (/ 54 2))) ;; result - 0,49999999999999994
```

```lisp
(sqrt (sq -5)) ;; result - 5
```
